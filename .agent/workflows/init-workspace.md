# Initialize Workspace

Set up the standard workspace directory structure.

## Commands

```bash
mkdir -p dev/workspace/{context,code,tests,docs}
mkdir -p dev/branches
echo "Workspace initialized: dev/workspace/ for active work, dev/branches/ for archives"
```

## Directory Structure

- `dev/workspace/context/` - PRDs, task lists, tree views, research notes
- `dev/workspace/code/` - Working code files
- `dev/workspace/tests/` - Test files
- `dev/workspace/docs/` - Documentation
- `dev/branches/` - Archived completed work

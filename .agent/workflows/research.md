# Research Workflow

Research external documentation, APIs, or technologies.

## Process

1. Identify what needs research
2. Search documentation/APIs
3. Document findings in `dev/workspace/context/research-[topic].md`
4. Include code examples and API endpoints

## Template

```markdown
# Research: [Topic]

## Summary
Brief overview of what was researched

## Key Findings
- Important discovery 1
- Important discovery 2

## Code Examples
[relevant code samples]

## API Endpoints
[if applicable]

## References
- [URLs and documentation links]
```

## Output

Research notes saved to `dev/workspace/context/research-[topic].md`
